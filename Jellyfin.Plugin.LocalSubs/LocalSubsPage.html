Local Subs : Subtitle provider for local subtitle files

Use directory separator native to Jellyfin's runtime ('\' on Windows, and '/' on Unix-like systems). Templates will be matched against the directory of target media (video).
Available placeholders are:
  • %f% : Filename with extension
  • %fn% : Filename without extension
  • %fe% : Filename extension
  • %l% : Language
  • %n% : Numbers
  • %any% : Any characters
